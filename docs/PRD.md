# PRD.md — AI Agent Skills Marketplace (v1.0)

> Product Requirements Document  
> SKILL.md 기반 AI 에이전트 스킬 카탈로그 & 마켓플레이스

---

## 1. 제품 개요

### 1.1 제품명
AI Agent Skills Marketplace

### 1.2 제품 한 줄 정의
SKILL.md 표준을 중심으로 전 세계 AI 에이전트 스킬을 수집·정규화·큐레이션하여  
사람과 에이전트가 **신뢰할 수 있는 스킬을 발견(discovery)** 할 수 있게 하는 플랫폼.

### 1.3 문제 정의 (Problem)
- Agent Skills(SKILL.md)가 여러 GitHub 리포, awesome 리스트, 개별 사이트에 **파편화**되어 있음
- 동일 스킬의 중복, 변종, 품질 편차로 인해 **탐색 비용이 높음**
- 인기/신뢰도/최근성 판단 기준이 없어 **선택이 어려움**
- 자동화만으로는 품질 관리가 어렵지만, 수작업만으로는 확장 불가

### 1.4 해결 방식 (Solution)
- 외부 생태계에서 스킬을 **자동 수집(raw)** 하고
- 운영자 승인 기반으로 **정식 스킬(skills)** 로 승격
- 표준화된 메타데이터, 인기 점수, 랭킹을 제공
- “Agent Skills Registry + Discovery Layer” 역할 수행

---

## 2. 목표와 성공 지표

### 2.1 제품 목표 (Goals)

1. SKILL.md 기반 스킬을 **중앙에서 탐색 가능**하게 한다
2. 스킬의 **품질·신뢰도·인기**를 빠르게 판단할 수 있게 한다
3. 신규 스킬 유입 → 정제 → 노출까지 **지속 가능한 파이프라인**을 만든다
4. IDE/에이전트 런타임과 연동 가능한 **확장 기반**을 마련한다

### 2.2 성공 지표 (KPIs)

- 활성 스킬 수 (Active skills)
- 주간 신규 스킬 승인 수
- 스킬 상세 페이지 조회수
- 즐겨찾기(favorite) 전환율
- TOP10 영역 클릭률
- 승인 대기(raw → skills) 평균 소요 시간

---

## 3. 사용자 정의

### 3.1 주요 사용자 유형

#### 1) 에이전트/자동화 사용자
- Claude / 커스텀 에이전트 / IDE 사용자
- 목적: “이 작업에 쓸 스킬이 뭐가 있지?”

#### 2) 스킬 제작자 (Creator)
- SKILL.md를 작성해 배포
- 목적: “내 스킬을 발견 가능하게 만들고 싶다”

#### 3) 운영자 (Admin/Curator)
- 품질 관리 및 중복 제거
- 목적: “자동화 + 사람 검증의 균형”

---

## 4. 핵심 사용자 시나리오 (User Stories)

### 4.1 스킬 탐색
- 사용자는 메인 페이지에서 “오늘의 인기 스킬 TOP 10”을 본다
- 카테고리 또는 태그로 스킬을 필터링한다
- 스킬 카드를 클릭해 상세 정보를 확인한다

### 4.2 스킬 평가
- 사용자는 다음 정보를 보고 스킬을 판단한다
  - 요약 설명
  - 최근 업데이트 시점
  - scripts 포함 여부
  - 라이선스
  - 전체/카테고리 랭킹

### 4.3 운영자 승인
- 운영자는 신규 raw 스킬 큐를 확인한다
- 잠재 중복 후보를 비교한다
- 새 스킬 등록 또는 기존 스킬에 병합한다

---

## 5. 기능 요구사항 (Functional Requirements)

### 5.1 스킬 카탈로그

- 스킬 목록 페이지
  - 페이지네이션
  - 정렬: 최신순 / 인기순
  - 필터: 카테고리 / 태그 / spec_format
- 스킬 상세 페이지
  - 정규화된 설명(Markdown)
  - 원본 리포/문서 링크
  - 스펙 형식(SKILL.md 여부)
  - 인기 지표 및 랭킹 배지

---

### 5.2 TOP 10 / 랭킹

- 전체 TOP 10
- 카테고리별 TOP 10
- 하루 단위 랭킹 스냅샷 유지
- 랭킹 기준:
  - GitHub stars
  - 내부 view/use/favorite
  - 최근성 가중치

---

### 5.3 크롤링 및 인입

- 대상:
  - Agiskills
  - SkillsMP / agskills.dev
  - Awesome Agent Skills
  - Anthropics skills 리포
- 기능:
  - 일 1회 이상 자동 수집
  - 변경 감지(commit SHA/ETag)
  - raw 데이터 보존

---

### 5.4 운영자 콘솔

- 신규 스킬 큐
- 파싱 상태(parse_status) 확인
- 잠재 중복 후보 제시
- 승인 액션:
  - 신규 등록
  - 기존 스킬 병합
  - 제외/보류
- 품질 체크:
  - scripts 포함 여부
  - 라이선스 신뢰도

---

### 5.5 트래킹 및 이벤트

- 이벤트 타입:
  - view
  - use (링크 클릭 기준)
  - favorite
- 개인정보 최소화:
  - session_hash
  - ip_hash
- 어뷰징 방지:
  - 동일 세션 반복 이벤트 제외

## 6. 비기능 요구사항 (Non-Functional Requirements)

### 6.1 확장성
- 소스 추가 시 크롤러 플러그인만 추가
- 스킬 수 증가에도 조회 성능 유지

### 6.2 안정성
- GitHub API 레이트리밋 대응(캐시/ETag)
- 원본 리포 삭제/비공개 시 tombstone 처리

### 6.3 보안
- scripts 포함 스킬은 기본 link_only 정책
- 악성 리포 블랙리스트 가능 구조

### 6.4 국제화
- 다국어 스킬 확장 가능 구조
- 언어 필드 분리 가능
